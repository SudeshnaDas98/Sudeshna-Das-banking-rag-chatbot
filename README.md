# AI Banking Support Chatbot

## Overview
This project is a RAG-based AI Banking Support Chatbot developed using Streamlit, LangChain, FAISS, and Google Gemini API.

## Features
- Banking chatbot
- RAG pipeline
- FAISS vector database
- Gemini API integration
- Streamlit deployment

## Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- Gemini API

## Deployment
Hugging Face Spaces
